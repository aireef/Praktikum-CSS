Font Style

CSS dapat mengatur style dari teks yang ada pada halaman web yang dibuat. terdapat beberapa property yang dapat digunakan untuk mengatur style teks yaitu sebbagai berikut

  • text-align: digunakan untuk mengatur perataan teks, value yang dapat digunakan yaitu left center, right
  • Color diigunakan untuk memberikan warna pada teks
  • letter-spacing: mengatur jarak antar karakter
  • word-spacing: mengatur jarak antar kata
